# doctor-appointment-rh

### `Setup`
```
git clone https://github.com/ItechIndrustries/doctor-appointment-rh.git
cd doctor-appointment-rh
virtualenv env
source env/bin/activate
pip3 install -r requirements.txt
python3 manage.py runserver
```
